## Folder tree explanation
- package.json - repo README summary and Playwright dependencies
- skills - Critical information about app architecture and purpose
- tests - Files used by me in original manual tests and Playwright for automated tests
	- prompts - Previous Chats with you involving this project
		- 0.35.18 - lowest number oldest → highest number newest
	- media - all files to be used by Playwright, or by me in manual tests. Streamlined from original "music" folder (removed) for cleaner test-files.js
	- helpers
		- index.js - starts local server from highest version of LineByLine
		- test-files.js - points to location of files for tests; customized by me from suggested example
	- \*spec.js - Playwright test files
	- \*snapshots - snapshot folders generated by the tests
- MANUAL.md - remaining manual tests *not* automated by Playwright
- CONTRIBUTING.md - tells potential contributors about Playwright tests
- CHECKLIST.md - deprecated lengthy manual tests that the Playwright tests are based on (added for clarity, will be removed in actual commit)
- HELP.md - user-facing explanation of how to use LineByLine
- linebyline-0.35.18.html - latest version of the app
- playwright.config.js - Playwright config
- .gitignore - snapshot folders are added for clarity, but hidden in the real commit
